# Bird Encyclopedia

Welcome to the comprehensive guide to birds! This documentation covers various species organized by their ecological groups.

## Categories

- **Songbirds** - Melodious passerines including robins, cardinals, and finches
- **Raptors** - Birds of prey such as eagles, hawks, and owls
- **Waterfowl** - Aquatic birds including ducks, geese, and swans
